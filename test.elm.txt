    Notes and hints

  * You can open/import the downloaded *.svg file with Inkscape
    <https://inkscape.org/> for further editing. Inkscape also allows
    you to export the equation as PDF document or PNG image.
  * In all modern browsers, you can insert SVG source code directly into
    a HTML document.
  * Rendering is performed within your browser. If you cache this page,
    you don't need an internet connection to render equations.
  * The equations are rendered using MathJax 3 <http://mathjax.org/>,
    which supports many TeX/LaTeX commands
    <http://docs.mathjax.org/en/latest/input/tex/macros/index.html>.
  * This has been tested with recent versions of Firefox and Chrome, but
    may work in other browsers as well.
  * With Chrome, you can add this as a web app to your home screen.
